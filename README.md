# E-Commerce-Sales-Dashboard-Power-BI
Interactive Power BI Dashboard to analyze E-commerce sales performance, revenue, trends, and key product diagnostics

## 🔑 Key Insights & Features
- **Total Revenue & Profit:** Tracks overall Sales (₹46M) and Net Profit (₹7.68M) with formatted currency metrics.
- **Order Profiling:** Analyzes total quantity sold (4,022 items) and distinct order counts (1,650 orders).
- **Time-Series Analysis:** A dynamic Area Chart showing monthly sales trends to identify peak performance months.
- **Geographical Breakdown:** Map visual depicting sales distribution across different states in India.
- **Product Diagnostics:** Bar charts detailing Sales by Category and uncovering the **Top 5 Sub-Categories** by sales.
- **Full Interactivity:** Implemented multi-select slicers for Category, City, State, and Payment Mode for seamless data drill-down.
